# Beca: AGCID Chile — Becas de Magíster República de Chile

## Datos Generales

| Campo | Valor |
|-------|-------|
| **Oferente** | Agencia de Cooperación Internacional para el Desarrollo de Chile (AGCID) |
| **Nombre oficial** | Programa de Becas de Magíster República de Chile |
| **Arquetipo** | Gobierno Bilateral |
| **Nivel** | Maestría (Magíster) |
| **Área** | Listado específico por convocatoria. **No se financian:** MBA ni especializaciones médicas. Se priorizan programas presenciales/híbridos. |
| **Modalidad** | Presencial, híbrida |
| **Idioma** | Español |
| **País de estudio** | Chile |
| **Financiamiento** | Parcial |
| **Grupo objetivo** | Profesionales activos en sector público y/o academia en su país de origen, con experiencia laboral relacionada con la maestría |

## Plazos

| Campo | Valor |
|-------|-------|
| **deadline_status** | closed |
| **deadline_typical** | Octubre del año anterior al inicio del programa |
| **Inicio del programa** | Generalmente marzo-abril (algunos en agosto) del año siguiente |
| **Según CSV ESPOL (2026)** | No hay registros recientes en el CSV de ESPOL para esta beca |
| **Ciclo 2026** | **CERRADO** (El plazo para iniciar en 2026 venció en octubre de 2025) |

> ⚠️ Los plazos para el programa principal de magíster suelen cerrar a principios de octubre del año anterior (ej. octubre 2025 para iniciar en 2026). Verifica en el portal oficial para la convocatoria 2027.

## Financiamiento — ¿Qué Cubre?

- Costo del arancel (matrícula)
- Costos de titulación
- Seguro médico
- Viático (estipendio mensual)
- Pasajes de avión (dependiendo de la convocatoria)

## Requisitos

1. Contar con título profesional de tercer nivel
2. Estar **actualmente trabajando** (en sector público o academia)
3. Carta de aceptación de una universidad chilena participante
4. Cumplir el perfil específico de la convocatoria del año

## Documentos Requeridos

| Documento | Notas ESPOL |
|-----------|-------------|
| Título universitario certificado | www.certificados.espol.edu.ec — $10 |
| Certificado de calificaciones | www.certificados.espol.edu.ec — $2 |
| Carta de aceptación universitaria chilena | Debe gestionarse directamente con la universidad en Chile **antes** de aplicar a la beca |
| Certificado de trabajo actual | De tu empleador o institución académica |
| CV en español | Formato estándar |

## Notas para Postulantes Ecuatorianos

- **Requisito crítico:** Debes estar trabajando actualmente — esta beca es para profesionales en ejercicio, no para recién graduados sin empleo formal.
- El proceso tiene dos pasos: primero obtener aceptación de una universidad chilena, luego postular a la beca.
- La relación bilateral Chile-Ecuador es sólida; hay cupos disponibles.
